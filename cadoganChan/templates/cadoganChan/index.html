Welcome!

Cadogan Chan is a free to use, no registration required, English language imageboard based on the Japanese imageboard Futaba Channel.

This imageboard is based primarily upon the posting of pictures and their discussion.

It's currently version 0.3.

Current Boards Available are:

  • testing

Currently Implemented Features:

  • Boards/Threads/Posts
  • Uploading Images
  • Sage
  • Noko
  • Tripcodes
  • Quoting Green Text with >
  • smart (cross-thread/board) referencing/replying with @
  • Administration Via Djangos admin panel

CadoganChan does not record the IP address of any of it's users though they are logged through apache. If privacy is a concern use a proxy (or tor) or don't post.

{% endblock %}

Pro Tips:

  • Image Types you can upload:
    • JPG
    • PNG
    • GIF
  • When starting a new thread you *MUST* have an image. Nothing else is neccesary.
  • When replying to a thread you must have an image and/or comment. nothing else is neccesary.
  • If you do not put a name you will be anonymous.
  • Putting 'sage' in the email field will stop the thread you're posting in from being bumped.
  • Putting 'noko' in the email field will return you to your new post.
  • A tripcode is a way to identify a user without registering by adding a unique hash to the end of their name. To use a tripcode put name!passphrase into the name field.
  • @<post_number> will allow you to refer to and link to any post, anywhere on the site. e.g. @12 will link to post 12.
